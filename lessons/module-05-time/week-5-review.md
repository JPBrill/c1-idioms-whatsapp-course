## 📱 Activity 2 — Correct the Mistakes

```
🧠 ACTIVITY 2: Spot the Error

Each sentence has an idiom used incorrectly. Find and fix the mistake.

1) "She made some last-minute changes after submitting the report."
2) "In the long run, I think we'll regret this decision next week."
3) "The exam is around the corner — it's still six months away."
4) "He decided to call it a day and continue working through the night."

Answers below ⬇️
```

---

## 📱 Activity 2 — Answers

```
✅ ANSWERS:
1) Error: you can't make last-minute changes AFTER submitting — should be BEFORE
2) Error: "in the long run" refers to the distant future, not "next week"
3) Error: "around the corner" = very soon, not six months away
4) Error: "call it a day" means STOP, not continue

How many did you catch? 🎯
```
