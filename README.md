# C1 Idioms WhatsApp Course

A thematic idioms course for C1 English learners, delivered as daily WhatsApp mini-lessons.

## How to use
- Send **1 lesson per day** in your WhatsApp group.
- Each lesson has **3 short messages** — send them one at a time.
- At the end of each week, send the **Weekly Review Activity**.
