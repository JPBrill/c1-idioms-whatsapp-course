## Exercise 4: Question Type Matching

Match each task description (1–6) to the correct question type (A–F).

**Descriptions:**
1. You read a paragraph and decide if it matches facts in the text, contradicts them, or isn't mentioned.
2. You choose the best answer from four options: A, B, C, or D.
3. You complete gaps in a short summary of a passage.
4. You decide if the paragraph heading best summarises the paragraph's main idea.
5. You label the different parts of a diagram based on the text.
6. You decide if a statement agrees with the writer's viewpoint.

**Question Types:**
- A. Multiple Choice
- B. True / False / Not Given
- C. Yes / No / Not Given
- D. Matching Headings
- E. Summary Completion
- F. Diagram Labelling
